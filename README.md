# ITSM Service Desk - Desktop Version 💻

This repository contains the core desktop-based architecture of the IT Service Management (ITSM) application. Developed as the first phase of the system's evolution, it implements the fundamental business logic, data models, and workflows for corporate ticket management.

## 🛠️ Tech Stack & Concepts
*   **Language:** Java (Core Java principles)
*   **UI Framework:** Java Swing / AWT
*   **Concepts:** Object-Oriented Programming (OOP), Layered Architecture, Local File I/O / In-Memory Data Management
